# Decentralized SSO Identity Management System

## Overview

This project implements a blockchain-based Identity and Access Management (IAM) system for secure Single Sign-On (SSO) authentication across multiple cloud applications. The system leverages Ethereum-based smart contracts, JWT token management, and role-based access control to provide decentralized, secure, and efficient authentication and authorization.

## Features

* Single Sign-On (SSO) authentication
* JWT-based secure token management
* Ethereum blockchain integration
* Smart contract-based access control
* Role-based authorization
* Secure file access and management
* Hash-based transaction verification
* Multi-application cloud access support

## System Modules
